# 侧边栏的顺序
# 数字越小越靠前，支持非整数和负数，比如 -10 < -9.5 < 3.2, order 为 -10 的文章会最靠上。
# 个人偏好将非干货或随想短文的 order 设置在 -0.01 到 -0.99，将干货类长文的 order 设置在 -1 到负无穷。每次新增文章都会在上一篇的基础上递减 order 值。
order: -0.01
---

在需要显示删除线的内容两边分别添加两个波浪号
```txt
~~这里是需要显示删除线的内容~~
```

显示效果
>~~这里是需要显示删除线的内容~~
